# 🔍 Searching

Searching is one of the fundamental concepts in Data Structures and Algorithms (DSA). It involves locating a specific element within a collection of data efficiently. Different searching techniques are used depending on the organization of data and the application's requirements.

This folder contains C++ mini projects that demonstrate the practical implementation of searching algorithms in real-world scenarios.

---

## 📂 Projects

### 🎓 Student Record Search System
A menu-driven application for managing student records. It demonstrates:
- Linear Search by Student Name
- Binary Search by PRN for update and deletion
- Record management using vectors
- Department-wise grouping using `map`

### 📚 Library Book Finder
A menu-driven application for managing library books. It demonstrates:
- Binary Search by Book ID
- Linear Search by Book Title
- Automatic sorting using `std::sort()`
- Dynamic record management using vectors
