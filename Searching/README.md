# 🔍 Searching

Searching is one of the fundamental concepts in Data Structures and Algorithms (DSA). It involves finding a specific element within a collection of data efficiently.

This folder contains C++ mini projects that demonstrate the practical implementation of searching techniques in real-world scenarios.

## 📂 Projects

- 🎓 Student Record Search System *(Completed)*
- 📚 Library Book Finder *(Coming Soon)*

## 📖 Searching Techniques Covered

- Linear Search
- Binary Search

## 🎯 Learning Objectives

- Understand when to use Linear Search and Binary Search.
- Analyze time complexities of different searching algorithms.
- Apply searching algorithms in practical applications.
- Build real-world projects using searching concepts.
