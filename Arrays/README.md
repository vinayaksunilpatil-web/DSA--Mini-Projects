# 📦 Arrays

Arrays are one of the fundamental data structures in Data Structures and Algorithms (DSA). They provide efficient storage and access to sequential data and serve as the foundation for many advanced algorithms and data structures.

This folder contains beginner-to-intermediate C++ mini projects that demonstrate the practical application of array-based concepts in real-world scenarios.

---

## 📂 Projects

### 📦 Inventory Management System
A menu-driven application for managing product inventory. It demonstrates:
- CRUD Operations
- Searching and Updating Records
- Merge Sort
- Dynamic data management using vectors

### 💰 Expense Tracker
A menu-driven application for managing personal expenses. It demonstrates:
- Expense categorization
- Budget tracking
- Merge Sort
- Hash Maps (`unordered_map`) for category-wise analysis
- Dynamic record management using vectors

---

## 📖 Concepts Covered

- Arrays using `vector`
- Searching
- Sorting (Merge Sort)
- Structures
- Classes & Objects
- Hash Maps (`unordered_map`)
- CRUD Operations
- Formatted Console Output

---

## 🎯 Learning Objectives

- Understand how arrays are implemented using vectors.
- Learn to perform CRUD operations on dynamic data.
- Apply searching and sorting algorithms in practical applications.
- Organize and analyze data using appropriate STL containers.
- Build real-world C++ applications using array-based concepts.
