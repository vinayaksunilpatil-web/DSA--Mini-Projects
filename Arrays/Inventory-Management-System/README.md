Inventory Mangemnt System

1) Problem Statemnt: Develop software to manage products in a store. The application should allow
   adding, updating, deleting, searching and sorting products while also identifying low - stock items.
   
3) Data Structures Used:
   -Struct
   -Vector
   
4) Algorithms used:
   -Merge Sort
   -Linear Search
   
5) Time Complexity Analysis:
   -Add : O(1)
   -Search :  O(n)
   -Sort(Merge Sort) : O(nlogn)
   -Budget Filter : O(n)
   -Delete : O(n)
   -Update : O(n)

6) Future Improments:
   GUI , DATABASE
